# Valkey - High-Performance Key-Value Store

## Overview

Valkey is a high-performance, open-source key-value store compatible with Redis protocols. Serves as the primary caching and data storage solution for various applications.

## Troubleshooting

1. **Persistence failures**

   - **Symptom**: Data loss after pod restarts
   - **Resolution**: Verify Rook Ceph storage provisioning and PVC binding
