## Guidelines

- Reference `.kilocode/context7-libraries.json` for a list of pre approved libraries to avoid having to call `resolve-library-id`
- Always check first if any predefined context7 library in `.kilocode/context7-libraries.json` is relevant to the topic at hand before answering and reference it if needed to reduce hallucinations and confirm before acting.
